## Draw grid
### PART A
Write a program than draws a MxN grid of the following unit square

```
+ - +
|   |
+ - +
```

for eg: 2x3 grid will look like
```
+ - + - + - +
|   |   |   |
+ - + - + - +
|   |   |   |
+ - + - + - +
```

### PART B
Write a program than draws a string of MxN chessboard

for eg: 2x3 grid will look like

```
+ - + - + - +
|   | B |   |
+ - + - + - +
| B |   | B |
+ - + - + - +
```
